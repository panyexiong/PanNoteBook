# 概述

## 数据结构的研究对象

- 数据间的逻辑结构
  - 集合：数据元素之间只有同属一个集合的关系
  - 线性关系：数据元素之间存在一个对一个的关系：
    - 对应java中的线性表：顺序表，链表，栈，队列。
  - 树形结构：数据元素之间存在一个对多个的关系：
    - 对应java中的树：二叉树
  - 网状结构（图形结构）：数据元素之间存在多个对多个的关系
    - 对应java中的图
- 数据的存储结构（或物理结构）

## 真实结构

- 线性表之顺序表（静态数据结构）：数组（Array）、ArrayList
- 线性表之链表（动态数据结构）：LinkedList

## 抽象结构（ADT）

- 栈stack
- 队列Queue
- 树Tree
- 图
- 其他
